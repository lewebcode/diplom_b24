# Термины и определения / Перечень сокращений и обозначений

*В настоящем отчёте применяют следующие термины с соответствующими определениями и сокращения.*

## Термины и определения

- **Платёжное решение** — юридическое лицо, сервис или банк, предоставляющие возможность проведения международных платежей (входящих и/или исходящих) для российских компаний (банки, платёжные шлюзы, финтех-платформы и т.п.).
- **Платёжный шлюз** — технологический сервис, обеспечивающий приём, обработку и проведение платёжных транзакций между участниками (например, между продавцом и покупателем, между банком и получателем перевода).
- **Интегральный индекс применимости** — сводный показатель, рассчитанный по системе критериев и весов, отражающий степень пригодности платёжного решения для заданного сценария (сфера торговли, страна, валюта, тип операций).
- **Мониторинг платёжных решений** — регулярный сбор и актуализация данных о платёжных решениях (тарифы, география, валюты, ограничения) из открытых и внутренних источников.
- **CRM-система** — информационная система для управления взаимоотношениями с клиентами и учёта сделок, в рамках которой может размещаться модуль анализа и рекомендаций по платёжным решениям.
- **ВЭД** — внешнеэкономическая деятельность.

## Перечень сокращений и обозначений

- **ВКР** — выпускная квалификационная работа  
- **API** — Application Programming Interface, программный интерфейс приложения  
- **БД** — база данных  
- **СППР** — система поддержки принятия решений  
- **B2B** — бизнес для бизнеса  
- **B2C** — бизнес для потребителя  
- **СБП** — система быстрых платежей  

*При необходимости список дополняется в тексте отчёта.*
